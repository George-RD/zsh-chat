# zsh-chat Vision & Ethos

## The Core Mission
To build a decentralized, sovereign social protocol for the terminal that rejects the "High-Fidelity Noise" of modern web platforms in favor of a "Lo-Fi High-Tech" aesthetic and mathematically grounded discovery.

## Evolutionary Mandate
This project is self-referential. In every development cycle (Ralph Loop), the agent must:
1. **Observe:** Analyze the current state of the TUI, the Relay, and the Social Graph.
2. **Question (Socratic):** Ask if the current implementation carries "Web2 baggage" that can be replaced with a more elegant terminal-native solution.
3. **Innovate:** Propose new features (e.g., 4-bit meme generators, tmux integrations, cryptographically-secure sub-groups) and document them as GitHub Issues.
4. **Refine:** Update this file and `product.md` as the "Truth" of the project evolves.

## The Ethos
- **Progressive Disclosure:** Simple for vibers, powerful for wizards.
- **Lo-Fi Constraint:** 4-bit/2-bit color is a feature, not a limitation. It neuters toxicity and shock-content.
- **Mathematical Discovery:** The Fibonacci Feed replaces the engagement algorithm.
- **Sovereignty:** You own your keys. You own your identity. You own your data.
